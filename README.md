## 사용기술
- RxSwift
- RxDataSource
- KingFisher
- MVVM
- Coordinator pattern
- DIContainer

- 캡처리스트
- UILable 줄바꿈 속성
- 강한 순환 참조
- ViewModel의 Input, Output, Action
- Content hugging priority, Compression resistance priority
